Res Partner Membership

Description

This module helps to configure Membership and manage Pricelist, Woo Membership on the partners based on the Membership. It also add a horizontal Ribbon on the Contact and its SO about the Membership.

Configuration

After installing the Res Partner Membership application, the feature will be enabled.

Go to Settings - - > Users & Companies - - > Users.

Click on the Admin user where the “ Can Create Membership ” boolean group is added under the access rights of the user.

Functionality

When the “ Can Create Membership ” boolean is enabled, a new field “Membership” is added in the contact form where we can select the type of Membership as shown below.

Upon selecting the Membership type, a green ribbon appears on top of the contact form with the selected membership name.

We can create a new Membership by selecting “New” and can select the “Pricelist” and “Woo Membership Name”.

After selecting the required Membership, create a sale order for the Membership contact where the Pricelist and the Green ribbon with the Membership name will be propagated from the contact form.

When the “ Can Create Membership ” boolean is disabled, we will not be able  to create a new membership, as the " New " button will be invisible.
